Acerca de Descripción del Proyecto

Esta aplicación está diseñada para aprender y practicar el lenguaje de señas, creada con el propósito de resolver un problema importante: las personas que no conocen el lenguaje de señas enfrentan dificultades para comunicarse de manera efectiva con la comunidad sorda, lo que limita la inclusión y la accesibilidad. La aplicación ofrece una herramienta educativa y de traducción en tiempo real para facilitar el aprendizaje del lenguaje de señas de manera accesible y práctica.
